# dgVoodoo 2 Bundle

[dgVoodoo 2](https://github.com/dege-diosg/dgVoodoo2) v2.87.1 by Dege.

Translates the game's 3Dfx Glide graphics calls to Direct3D 11, enabling
windowed mode and Alt+Enter fullscreen toggle on modern Windows.

Auto-installed into the game folder by the trainer on first run.
Original files are backed up to `*.bak_nglide`.

**Freeware** — see [upstream](https://dege.freeweb.hu/dgVoodoo2/) for terms.
